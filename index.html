Erreur de redirection : Si vous tombez sur cette page, c'est que la redirection n'est pas bonne ou que l'application n'est pas installée.
Solution 1. Veuillez configurer votre virtualhost afin qu'il pointe vers le dossier public de l'application.

Solution 2. Vérifier que le fichier .htaccess est présent à la racine de Proaxive.
