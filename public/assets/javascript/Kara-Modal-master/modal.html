Kara Modal Box v1.1

Kara est une boite modal développé en javascript sans framework

Télécharger Kara Modal Box v1.1
Kara permet de mettre en place un système de modale simple (sans jQuery), accessible et compatible avec la navigation au clavier.
Utilisation

Chargez le style CSS de Kara (avant la fermeture de la balise <head>)

<link rel="stylesheet" href="path/css/kara.css">

Chargez le fichier javascript de Kara (avant la fermeture de la balise <body>)

<script src="path/js/kara.js"></script>
Modale simple

Création d'un lien avec l'ID modal1 et la class js-kara-modal

<a href="#modal1" class="js-kara-modal">Ouvrir la boite modale</a>
Modal avec entête et footer Modal sans entête et footer
Modale avec Ajax
Important : il n'est pas possible d'afficher une page externe

Création d'un lien avec l'ID modal2 (vers le fichier modalAjax.html) et la class js-kara-modal

<a href="modalAjax.html#modal2" class="js-kara-modal">Ouvrir avec Ajax</a>
Ouvrir en Ajax
Kara et Animate.css

Vous pouvez également utiliser la librairie CSS Animate afin d'aller encore plus loin dans l'animation de la boite modale.
Après avoir chargé animate.css, ajoutez la class animated suivi de l'animation souhaitée.

<div class="modal-wrapper animated swing js-kara-modal-stop">
